# SOP OperASIONAL SISTEM MES

## (Work Order, Produksi, OEE, dan Monitoring Mesin)

---

# 1. TUJUAN

SOP ini bertujuan untuk:

* Menstandarkan penggunaan sistem MES di area produksi
* Memastikan data produksi akurat dan real-time
* Mengurangi kesalahan input oleh operator
* Mendukung monitoring performa mesin (OEE)

---

# 2. RUANG LINGKUP

SOP ini berlaku untuk:

* Operator produksi
* Supervisor produksi
* Admin sistem

Area:

* Semua mesin produksi yang terdaftar dalam sistem

---

# 3. DEFINISI

* **MES (Manufacturing Execution System)**: Sistem untuk monitoring dan pencatatan aktivitas produksi
* **Work Order (WO)**: Perintah produksi
* **OEE**: Overall Equipment Effectiveness
* **Downtime**: Waktu mesin berhenti
* **Reject**: Produk gagal

---

# 4. TANGGUNG JAWAB

## 4.1 Operator

* Input produksi
* Input reject
* Input downtime
* Memastikan data yang diinput akurat

## 4.2 Supervisor

* Monitoring dashboard
* Verifikasi data produksi
* Analisa performa mesin

## 4.3 Admin

* Maintain sistem
* Manage user
* Backup data

---

# 5. PROSEDUR OPERASIONAL

## 5.1 Login Sistem

1. Operator membuka sistem MES
2. Masukkan username dan password
3. Klik tombol LOGIN
4. Jika gagal, hubungi admin

---

## 5.2 Pemilihan Work Order

1. Pilih mesin pada dashboard
2. Klik menu Work Order
3. Pilih WO yang sesuai
4. Klik SET WO
5. Pastikan WO aktif sebelum produksi

---

## 5.3 Input Produksi

1. Klik tombol Input Produksi
2. Masukkan jumlah produksi (good)
3. Klik SIMPAN
4. Pastikan data sesuai dengan kondisi aktual

---

## 5.4 Input Reject

1. Klik tombol Reject
2. Masukkan jumlah reject
3. Pilih alasan reject
4. Klik SIMPAN

---

## 5.5 Input Downtime

1. Klik tombol Downtime
2. Pilih alasan downtime
3. Klik START DOWNTIME
4. Setelah selesai, klik END DOWNTIME

---

# 6. ATURAN PENGGUNAAN

* Input data harus dilakukan secara real-time
* Dilarang mengisi data secara estimasi
* Setiap downtime wajib dicatat
* Setiap reject wajib dicatat dengan reason
* Tidak diperbolehkan mengganti WO tanpa izin

---

# 7. KONTROL DAN VALIDASI

* Sistem akan memvalidasi input
* Supervisor melakukan pengecekan harian
* Data produksi dibandingkan dengan hasil fisik

---

# 8. PENANGANAN MASALAH

## 8.1 Sistem Tidak Bisa Diakses

* Hubungi admin IT

## 8.2 Data Salah Input

* Laporkan ke supervisor
* Lakukan koreksi sesuai prosedur

## 8.3 Downtime Tidak Dicatat

* Segera update di sistem
* Berikan catatan tambahan

---

# 9. MONITORING DAN EVALUASI

* Supervisor memonitor dashboard setiap shift
* Evaluasi OEE dilakukan harian/mingguan
* Analisa downtime untuk improvement

---

# 10. PENUTUP

SOP ini wajib dipatuhi oleh seluruh pengguna sistem MES.

Pelanggaran terhadap SOP dapat mempengaruhi akurasi data produksi dan evaluasi kinerja.
