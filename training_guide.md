# Training Guide & Simulasi Penggunaan Sistem MES

## Untuk Onboarding Operator Baru

---

# 1. Tujuan Training

Dokumen ini digunakan untuk:

* Membantu operator baru memahami sistem MES
* Memberikan simulasi penggunaan secara bertahap
* Mengurangi kesalahan saat penggunaan pertama

---

# 2. Durasi Training

Total estimasi: **2 – 3 jam**

| Sesi | Materi                     | Durasi   |
| ---- | -------------------------- | -------- |
| 1    | Pengenalan sistem          | 30 menit |
| 2    | Dashboard & navigasi       | 30 menit |
| 3    | Simulasi input produksi    | 45 menit |
| 4    | Simulasi downtime & reject | 45 menit |
| 5    | Review & evaluasi          | 30 menit |

---

# 3. Sesi 1 – Pengenalan Sistem

## Materi:

* Apa itu MES
* Fungsi sistem
* Pentingnya input data real-time

## Penjelasan sederhana:

```text
MES = sistem untuk mencatat dan memonitor produksi
```

---

# 4. Sesi 2 – Dashboard & Navigasi

## Tujuan:

Operator dapat membaca dashboard

## Simulasi:

1. Login ke sistem
2. Lihat dashboard mesin
3. Identifikasi:

   * Mesin running
   * Mesin downtime
   * Mesin off

## Latihan:

* Trainer menunjuk 1 mesin
* Operator diminta menjelaskan statusnya

---

# 5. Sesi 3 – Simulasi Input Produksi

## Tujuan:

Operator bisa input produksi dengan benar

## Simulasi:

```text
Step 1: Klik mesin
Step 2: Lihat WO aktif
Step 3: Klik Input Produksi
Step 4: Input qty (misal: 100)
Step 5: Klik SIMPAN
```

## Latihan:

* Input 100 pcs
* Input 50 pcs
* Gunakan tombol +10 / +50

## Evaluasi:

* Apakah jumlah tersimpan dengan benar?
* Apakah operator memahami flow?

---

# 6. Sesi 4 – Simulasi Reject

## Tujuan:

Operator bisa input reject dengan benar

## Simulasi:

```text
Step 1: Klik Reject
Step 2: Input qty (misal: 5)
Step 3: Pilih reason
Step 4: Klik SIMPAN
```

## Latihan:

* Input reject 3 pcs
* Pilih reason yang sesuai

## Evaluasi:

* Apakah reason dipilih dengan benar?

---

# 7. Sesi 5 – Simulasi Downtime

## Tujuan:

Operator bisa mencatat downtime

## Simulasi:

```text
Step 1: Klik Downtime
Step 2: Pilih reason
Step 3: Klik START
Step 4: Tunggu 1 menit
Step 5: Klik END
```

## Latihan:

* Simulasikan downtime karena "Call Mechanic"

## Evaluasi:

* Apakah downtime tercatat?
* Apakah durasi muncul?

---

# 8. Sesi 6 – Work Order

## Tujuan:

Operator memahami WO aktif

## Simulasi:

```text
Step 1: Klik Work Order
Step 2: Pilih WO baru
Step 3: Klik SET WO
```

## Catatan:

* Tidak semua operator boleh ganti WO

---

# 9. Simulasi Lengkap (End-to-End)

## Scenario:

```text
1. Login
2. Pilih mesin
3. Set WO
4. Input produksi (100 pcs)
5. Input reject (5 pcs)
6. Start downtime
7. End downtime
8. Kembali ke dashboard
```

---

# 10. Checklist Kompetensi Operator

Operator dianggap siap jika:

* Bisa login
* Bisa membaca dashboard
* Bisa input produksi
* Bisa input reject
* Bisa input downtime
* Tidak melakukan kesalahan fatal

---

# 11. Kesalahan yang Harus Dihindari

* Tidak set WO sebelum produksi
* Lupa stop downtime
* Input jumlah tidak sesuai
* Tidak memilih reason

---

# 12. Tips Training

Untuk trainer:

* Gunakan contoh nyata
* Biarkan operator mencoba langsung
* Jangan terlalu banyak teori
* Fokus ke praktik

Untuk operator:

* Jangan ragu bertanya
* Fokus pada alur
* Latihan berulang

---

# 13. Penutup

Training ini bertujuan agar operator dapat menggunakan sistem MES dengan cepat dan benar.

Semakin sering digunakan, operator akan semakin terbiasa dan kesalahan akan berkurang.
